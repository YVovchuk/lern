Запись на обучение

Выберите дату, преподавателя и свободное окно. Все записи сохраняются в общей базе, поэтому слоты будут одинаковыми для всех пользователей.

Нажмите на слот, чтобы выбрать время. Занятое окно становится недоступным для других сразу после сохранения в базе.

Как это работает

Для каждого сочетания дата + учитель + время создаётся одна запись в Firestore. Если слот уже занят, он блокируется для всех пользователей.

Кнопка «Посмотреть все записи» открывает список всех бронирований: кто записан, на какое время и к какому учителю.
